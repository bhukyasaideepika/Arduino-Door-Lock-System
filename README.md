# Arduino Door Lock System

## Overview

Designed and implemented a password-protected electronic door lock system using Arduino. The project demonstrates embedded systems programming, sensor interfacing, and secure access control.

## Technologies Used

* Arduino
* Embedded C
* Keypad
* Servo Motor
* LCD Display

## Features

* Password authentication
* Secure door access
* LCD status display
* Incorrect password detection

## Skills Demonstrated

* Embedded Systems
* Arduino Programming
* Hardware Interfacing
* Problem Solving
